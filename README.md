# Introduction to Computer Architecture

> Students should master the basic concepts and general operation of a computer, the physical structure
> supporting the execution of programs. They should be able to describe the operation of the basic
> elements of a computer: the processor, the memory system and the input/output system, and their
> interactions. In particular, they should be able to understand the internal structure of a processor
> and the process of instruction execution.

## Project 1 - Neural Network Classifier

### Goal
Implement the core arithmetic and neural‑network support routines for a handwritten‑digit classifier
entirely in RISC‑V Assembly. The project focuses on low‑level computation, memory manipulation, and
control‑flow design, building up from integer arithmetic to the foundations of a simple neural‑network
inference pipeline. It reinforces a deep understanding of the RISC‑V Instruction Set Architecture
(ISA), stack order, calling conventions, and efficient register usage with computer architecture and
machine‑learning concepts.

## Project 2 - Neural Network Inference Engine

### Goal
Extend the neural‑network classifier by implementing the full forward‑propagation pipeline in RISC‑V
Assembly. This includes matrix–vector multiplication, bias addition, activation functions, and
layer‑by‑layer computation to classify handwritten digits. The project deepens understanding of the
stack, memory layout, performance‑aware loop design, and expressing high‑level numerical algorithms in
a low‑level language.

## Project 3 - 8‑bit Spike Processor

### Goal
Design and implement an 8‑bit processor using Logisim, capable of executing a custom instruction set
including arithmetic operations and AI‑oriented instructions. This project consolidates digital‑logic
concepts by integrating registers, ALU components, control logic, and a ROM‑based program memory into
a functional CPU. It emphasizes architectural trade‑offs, instruction encoding, and the practical
challenge of building a full working processor.
